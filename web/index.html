04 · Tree Anatomy

B+ Tree 해부.

레코드 수 (N)
분기 수 (Order)
레코드 수 (N) — 트리에 삽입할 id 개수. 1..N 을 랜덤 순서로 넣어 실제 split 이 발생하는 모습을 보여준다. 기본 20. 최대 200 (화면 폭 제한).
분기 수 (Order) — 한 노드가 가질 수 있는 최대 자식 개수. 작을수록 (4) 트리가 금방 깊어져 split 구조가 잘 보이고, 클수록 (128) 실제 프로덕션처럼 납작해진다.
